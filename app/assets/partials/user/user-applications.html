My applications

Programmatic access can be handy or necessary in different scenarios, for example:

  • You would like to export regularly the PTMs of a proteins list you have previously saved (See this QA on how to do this)
  • You are building a javascript website (be careful with your API key, see the recommendations in our QA)
  • You would like to access our SPARQL endpoint to make federated queries

If you would like to access the API or SPARQL endpoint programatically please create an application by providing a meaningful name and description.

All API are limited to 1000 calls/day. If you need more than this, please contact us or provide a good explanation in the description.
